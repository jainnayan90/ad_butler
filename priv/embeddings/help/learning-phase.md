# Learning phase

When you create or significantly edit an ad set, Meta restarts a learning
phase to gather conversion data. The ad set is in learning until it
accumulates roughly 50 optimization events within seven days.

Performance during learning is unstable — CPA tends to be high and CTR
swings wildly. Resist the urge to tweak budgets or audiences mid-learning;
each significant edit restarts the phase.

A `stalled_learning` finding indicates an ad set has been in learning for
more than seven days without exiting. Common fixes: widen the audience,
raise the daily budget, or change the optimization event to one that
fires more often.
